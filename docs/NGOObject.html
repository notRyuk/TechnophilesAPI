new NGOObject(id, name, startTime, closeTime, days, latitude, longitude, phone, email, line1Address, line2Address, cityOrVillage, state, pinCode)

Source:
  • db.js, line 847

The main class for all the NGO routes

Parameters:
Name Type Description
id String

The ID of the NGO in the database or that is to be created
name String

The name of the NGO in the database or that is to be created
startTime String

The daily start timings of the NGO in the format of HH:mm
closeTime String

The daily close timings of the NGO in the format of HH:mm
days String

The days of the week in which the NGO operates in the format of Mon-Sat (case sensitive)
latitude Number

The latitude of the NGO's location which can be accurate upto 50m
longitude Number

The longitude of the NGO's location which be accurate upto 50m
phone String

The mobile or telephone number of the NGO which is used to be contacted with (Default Format: +919999999999)
email String

The email ID of the NGO in the database or that is to be added in the database
line1Address String

The line 1 address of the NGO required
line2Address String

The line 2 address of the NGO (This param is optional [For optional leave an empty string])
cityOrVillage String

The nearest city or village that the NGO is located in
state String

The state in which the NGO is located in
pinCode Number

The pin code of the location in which the NGO is located
